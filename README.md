# Todo list application repository

Frontend project of a todo list inspired in spacial missions. \
The project uses localSotrage as first phase for store your own tasks, by the time will use a external DB connected within an API.\

Feel free to clone this project if you found it useful git clone.

## Technologies
This project was developed with my knowledge and powered by AI (Claude Haiku).
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

I strongly recommend to use pnpm instead of npm, so use the following commands to install and use this application.

### `pnpm install`

Will install all the dependencies that application uses.

### `pnpm run start`
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.

### `pnpm run deploy`

Builds the app for github pages, be sure that you have everything set in your version to use GitHub pages\
